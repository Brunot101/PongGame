# Descrição
Tentativa de recriar o clássico jogo de Atari "Pong"
# Como executar a aplicação
Basta abrir o arquivo index.html . O jogador 1 é controlado pelas teclas W e S, enquanto o jogador 2 utiliza as setas para cima e para baixo.Em caso de modo coop as teclas F e R movimentam uma das barras, e L e O a outra
